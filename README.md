# clase-fundepos

Curso de modelación

## Descripción

Este proyecto inicial introduce la sintaxis básica de Python. Incluye ejemplos simples de código y un notebook interactivo.

## Archivos

- `primeraclase.py`: Script básico que imprime "hola clase".
- `elarchivo2delcurso.ipynb`: Notebook con ejemplos de código y listas en Markdown.

## Cómo ejecutar

Para ejecutar el script Python:
```bash
python primeraclase.py
```

Para el notebook, ábrelo en Jupyter o VS Code y ejecuta las celdas.
